# 灵动转图王 (HeicConverter) 用户使用手册

感谢您购买本软件！这是一份详细的操作指南，帮助您在不同设备上高效、安全地使用本工具。

---

## 🌟 核心特色
*   **极致隐私**：本软件采用纯本地转换技术。您的照片**永远不会上传到任何服务器**。
*   **离线可用**：无需互联网连接，在断网状态下依然可以完美运行。
*   **全端支持**：包含 Windows 桌面版和单文件网页版，覆盖电脑、iPhone、安卓全平台。
*   **保留元数据**：自动保留照片的拍摄时间、地点（EXIF）等原始信息。

---

## 1. Windows 桌面版使用指南 (`HeicConverter.exe`)
*适合 Windows 10/11 系统，支持高性能批量转换。*

### 🚀 启动说明
1.  双击运行 `HeicConverter.exe`。
2.  **重要提示**：由于本工具为独立开发作品，未购买微软昂贵的数字证书，初次运行时 Windows 可能会弹出蓝色保护窗口。
    *   **操作方法**：点击窗口中的【更多信息】 -> 点击底部出现的【仍要运行】即可进入。请放心，本程序无任何恶意代码。

### 🛠️ 转换步骤
1.  **添加照片**：直接将 HEIC/HEIF 照片拖入软件窗口，或点击中心区域选择文件。
2.  **调整设置**：
    *   **格式**：支持导出为 JPG、PNG、WebP。
    *   **质量**：滑动滑块调节。建议 85%-90% 以获得最佳画质与体积平衡。
3.  **下载保存**：
    *   点击单张图片下方的“下载”图标保存。
    *   点击顶部的【全部打包下载】可一键生成 ZIP 压缩包保存所有照片。

## 2. 单文件网页版使用指南 (`index.html`)
*这是一项“黑科技”封装，将整个转换引擎浓缩在一个 HTML 文件中，方便随身携带。*

### 💻 电脑端 (PC/Mac)
1.  使用任意浏览器（推荐 Chrome, Edge, Safari）打开 `index.html`。
2.  操作方式与桌面版完全一致：拖拽 -> 转换 -> 下载。

### 📱 苹果设备 (iPhone/iPad)
1.  **保存**：将文件发送到手机，保存至【文件】App 中。
2.  **运行**：在【文件】App 中直接点击打开。
3.  **进阶用法 (推荐)**：
    *   用 Safari 浏览器打开该文件。
    *   点击底部的“分享”按钮 -> 选择【添加到主屏幕】。
    *   现在它会像原生 App 一样出现在桌面上，且支持离线全屏使用。

### 🤖 安卓设备 (Android)
1.  使用手机自带浏览器或 Chrome 打开该文件。
2.  点击上传区域，从相册中选取照片即可开始。

## 3. 常见问题 (FAQ)

**Q：转换后的照片保存在哪里？**
*   **网页版**：保存在浏览器的默认“下载”文件夹中。
*   **桌面版**：下载单张或打包时会弹出窗口，由您决定保存位置。

**Q：为什么有的照片转换失败？**
*   请确保原始文件是标准的 `.heic` 或 `.heif` 格式。如果是从损坏的存储卡中提取的残缺文件，可能会报错。

**Q：软件真的不联网吗？**
*   您可以完全断开 Wi-Fi 或拔掉网线进行测试。本软件的转换逻辑全部运行在您的设备内存中，绝不产生上传流量。
